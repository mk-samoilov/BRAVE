# Модуль brv_math

Обёртка-реэкспорт над крейтом `glam`. Никакой собственной логики — просто делает типы glam доступными через единый импорт.

---

## Реэкспорт

```rust
pub use glam::{Vec2, Vec3, Vec4, Mat4, Quat};
```

### Типы

| Тип | Описание |
|---|---|
| `Vec2` | 2D вектор (x, y) — UV координаты, позиция на экране |
| `Vec3` | 3D вектор (x, y, z) — позиции, направления, масштаб |
| `Vec4` | 4D вектор (x, y, z, w) — гомогенные координаты |
| `Mat4` | 4×4 матрица — трансформации, проекции |
| `Quat` | Кватернион — вращения без gimbal lock |

### Почему glam

- SIMD-оптимизация (SSE2/SSE4.1/AVX2 на x86, NEON на ARM)
- Zero-cost abstractions — типы по размеру равны raw массивам
- `#[repr(C)]` — совместимость с GPU uniform buffers
- Стандарт в Rust gamedev (используется в Bevy, Fyrox, rend3)
